## What Converts Well (and What Doesn't)

### ✅ Converts perfectly

| Markdown Element | PowerPoint Result |
|-----------------|-------------------|
| `# Heading 1` | Title slide |
| `## Heading 2` | Section slide |
| `### Heading 3` | Sub-section slide |
| `- Bullet lists` | Bullet points |
| `1. Numbered lists` | Numbered items |
| `| Tables |` | PowerPoint tables |
| `**Bold**` | Bold text |
| `*Italic*` | Italic text |
| `` `code` `` | Monospaced text |

### ⚠️ Needs manual adjustment

| Element | What happens | What to do |
|---------|-------------|------------|
| Images (`![alt](url)`) | Placeholder or skipped | Add images manually in PowerPoint |
| Complex nested lists | May flatten | Adjust indentation in PowerPoint |
| Code blocks | Rendered as plain text | Use a monospace font |
| Horizontal rules (`---`) | May or may not convert | Add divider lines in PowerPoint |

### 💡 Pro tip

Before converting, paste your Markdown into the **[Markdown Editor](/tools/markdown-editor)** to preview exactly how it looks. Fix any formatting issues there first — it's faster than fixing them in PowerPoint.
