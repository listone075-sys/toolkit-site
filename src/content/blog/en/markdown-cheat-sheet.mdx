# Markdown Cheat Sheet — Complete Quick Reference Guide

Markdown is everywhere in 2026. **ChatGPT and Claude output Markdown.** GitHub renders it. Slack, Notion, and Discord all support it. If you work with text online, you use Markdown — whether you realize it or not.

This cheat sheet covers **every Markdown syntax** you'll ever need: basic formatting, extended GFM (GitHub Flavored Markdown), tables, code blocks, footnotes, emoji, and escape characters.

## Basic Syntax

### Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

### Text Formatting

```markdown
**bold text**
*italic text*
***bold italic text***
~~strikethrough~~
==highlighted text== (mark)
<ins>underlined</ins> (HTML)
```

### Links

```markdown
[Basic link](https://example.com)
[Link with title](https://example.com "Click me")
<https://automatic-link.com>
[Reference-style][ref]

[ref]: https://example.com
```

### Images

```markdown
![Alt text](image.jpg)
![Alt text with title](image.jpg "Photo caption")

![Reference image][img-ref]
[img-ref]: image.jpg
```

### Blockquotes

```markdown
> Single line quote

> Multi-line
> paragraph quote
>
> — Author

> Nested
>> blockquote
```

### Horizontal Rules

```markdown
---
***
___
```

## Lists

### Unordered Lists

```markdown
- Item one
- Item two
  - Nested item (2 spaces indent)
  - Another nested
- Item three
```

You can also use `*` or `+` interchangeably:

```markdown
* First level
  + Second level
    - Third level
```

### Ordered Lists

```markdown
1. First item
2. Second item
   1. Nested ordered
   2. Another nested
3. Third item
```

Start at any number:

```markdown
57. Start from 57
58. Continues from here
```

### Task Lists (GFM)

```markdown
- [ ] Not started
- [x] Completed
- [ ] ~~Cancelled task~~
```

Result:

- [ ] Not started
- [x] Completed
- [ ] ~~Cancelled task~~

## Tables

```markdown
| Header 1 | Header 2 | Header 3 |
|----------|:--------:|---------:|
| Left     | Center   | Right    |
| Data     | Data     | Data     |
```

Result:

| Header 1 | Header 2 | Header 3 |
|----------|:--------:|---------:|
| Left     | Center   | Right    |
| Data     | Data     | Data     |

**Alignment syntax:**
- `:---` — left-aligned
- `:---:` — center-aligned
- `---:` — right-aligned

## Extended Syntax

### Footnotes (GFM)

```markdown
Here's a sentence with a footnote.[^1]

[^1]: This is the footnote content.
```

### Strikethrough (GFM)

```markdown
~~This text is crossed out.~~
```

### Emoji Shortcodes

```markdown
:rocket: → 🚀
:fire: → 🔥
:warning: → ⚠️
:white_check_mark: → ✅
:bookmark: → 🔖
```

### Automatic URLs

```markdown
<https://toolcraftbox.com>
<email@example.com>
```

### Heading IDs

```markdown
## My Section {#custom-id}
```

Link to it:

```markdown
[Jump to section](#custom-id)
```

### Definition Lists

```markdown
Term
: Definition line 1
: Definition line 2
```

### Superscript & Subscript

```markdown
X^2^ (superscript)
H~2~O (subscript)
```

### Escape Characters

Use backslash to display literal markdown characters:

```
\* Not italic \*
\# Not a heading
\- Not a list
\` Not code
\[ Not a link
```

## HTML in Markdown

Markdown supports raw HTML for extended formatting:

```html
<details>
<summary>Click to expand</summary>
Hidden content here.
</details>

<kbd>Ctrl</kbd> + <kbd>C</kbd>

<p style="color: red;">Red text</p>
```
