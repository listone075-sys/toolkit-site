## 为什么 Markdown 转 PPT 是效率革命

传统的幻灯片制作非常耗时。打开 PowerPoint，选择模板，逐页输入标题，复制粘贴要点，格式化表格，调整对齐……

有了 Markdown 转 PPTX 转换，结构已经就绪。Markdown 几乎是为幻灯片大纲而生的：

```markdown
# 幻灯片标题

- 要点一
- 要点二
- 要点三

| 指标 | Q1 | Q2 |
|------|----|----|
| 收入 | 10万 | 15万 |
```

每个 `#` 标题就是一页幻灯片。每个 `-` 列表就是演讲要点。每个 `|` 表格就是数据页。

**你专注于内容。格式交给转换器。**

## 什么能完美转换（什么需要手动调整）

### ✅ 完美转换

| Markdown 元素 | PowerPoint 结果 |
|-------------|----------------|
| `# 一级标题` | 封面页 |
| `## 二级标题` | 章节页 |
| `### 三级标题` | 子章节页 |
| `- 无序列表` | 要点列表 |
| `1. 有序列表` | 编号列表 |
| `| 表格 |` | PowerPoint 表格 |
| `**加粗**` | 加粗文字 |
| `*斜体*` | 斜体文字 |

### ⚠️ 需要手动调整

| 元素 | 情况 | 处理方法 |
|------|------|---------|
| 图片 (`![alt](url)`) | 可能跳过 | 在 PowerPoint 中手动添加 |
| 复杂嵌套列表 | 可能展平 | 在 PowerPoint 中调整缩进 |
| 代码块 | 渲染为纯文本 | 手动设为等宽字体 |
