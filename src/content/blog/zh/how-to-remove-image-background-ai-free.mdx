## AI 背景去除的工作原理

这并非魔法——而是一个经过数百万张图片训练的神经网络，学会了识别什么是"主体"、什么是"背景"。

### 技术原理（简化版）

1. **你上传一张图片** — 产品照、人像、logo，任何内容
2. **AI 模型加载** — 约 40MB，首次下载后缓存在浏览器中
3. **像素级分割** — 模型将每个像素分类为前景或背景
4. **背景已移除** — 提取主体，透明 PNG 即可下载

我们使用的模型是 **ISNet**（图像分割网络），通过 **ONNX Runtime Web** 以 WebAssembly 和 GPU 加速运行。大多数图片处理时间为 2-10 秒。

### 为什么浏览器端处理很重要

大多数背景去除工具的工作方式：

```
你的图片 → 上传到远程服务器 → AI 处理 → 下载结果
```

浏览器端处理的工作方式：

```
你的图片 → AI 在你的设备上运行 → 下载结果
```

你的图片**从不会离开你的设备。** 对于处理敏感照片的任何人——产品原型、个人肖像、机密文件——这就是安全与风险的区别。

## 获得最佳效果的小贴士

### 1. 确保良好对比度

AI 在主体与背景对比明显时效果最佳。站在杂乱花纹墙前的人比站在纯色墙前的人更难处理。

### 2. 良好光线有帮助

曝光均匀、光线充足的照片边缘更干净。避免刺眼的阴影让主体融入背景。

### 3. 头发和毛发：调整预期

AI 模型已有巨大进步，但精细的发丝和毛发边缘仍然是最难处理的场景。如果单根发丝的细节很重要，可能仍需 Photoshop 手动处理。

### 4. 纯色背景最简单

如果专门为去背景拍照，使用纯色背景（任何颜色——只要与主体颜色不同即可）。AI 更容易将主体与背景分离。

### 5. 下载前检查边缘

放大并仔细检查结果边缘。如果看到粗糙边缘，试试对比度更好的输入图片。输出质量很大程度上取决于输入质量。
