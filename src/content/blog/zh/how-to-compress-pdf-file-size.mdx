# 如何压缩 PDF 文件大小

你需要通过邮件发送一份 PDF，但它有 15MB，而邮件附件限制是 10MB。或者某个表单提示"最大文件：5MB"。以下是如何缩小它的方法 — 无需安装软件。

## PDF 为什么这么大

- **嵌入的图片** — 高分辨率照片会让 PDF 臃肿不堪
- **未优化的字体** — 嵌入的字体文件会增加数 MB
- **扫描页面** — 每页本质上就是一张全尺寸图片
- **未使用的对象** — 编辑历史记录和元数据

## ToolCraft 方法（免费、无需上传）

1. 使用 [PDF 转 JPG](https://toolcraftbox.com/tools/pdf-to-jpg) 提取所有页面为图片
2. 通过 [图片压缩器](https://toolcraftbox.com/tools/image-compressor) 压缩这些图片
3. 用 [JPG 转 PDF](https://toolcraftbox.com/tools/jpg-to-pdf) 重新构建 PDF
4. 如果有多个 PDF，使用 [合并 PDF](https://toolcraftbox.com/tools/merge-pdf) 将它们合并

## 压缩效果预期

| 原始大小 | 压缩后 | 节省 |
|----------|--------|------|
| 15 MB | 3–5 MB | 67–80% |
| 5 MB | 1–2 MB | 60–80% |
| 1 MB | 300–500 KB | 50–70% |

## 专业技巧

- **不要过度压缩** — 低于 150 DPI，文字会变得难以阅读
- **检查结果** — 务必打开压缩后的 PDF 验证可读性
- **考虑拆分** — 如果一份 PDF 太大，只提取需要的页面
