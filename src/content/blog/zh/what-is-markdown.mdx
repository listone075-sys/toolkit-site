# 什么是 Markdown？初学者指南（2026）

Markdown 是一种轻量级标记语言，用于格式化文本。它由 John Gruber 和 Aaron Swartz 于 2004 年创建，目标是让文本在**不依赖复杂工具**的情况下易于编写和阅读。

## 为什么 Markdown 在 2026 年如此重要

如今，Markdown 已成为 AI 聊天输出的通用语言。ChatGPT、Claude、Gemini 等 AI 工具以 Markdown 格式输出格式化文本。开发者使用 Markdown 编写 README 和技术文档。写作者利用 Markdown 的简洁性进行内容创作。

## Markdown 快速入门

```markdown
# 这是标题
## 这是副标题
**这是粗体文本**
*这是斜体文本*
- 列表项
- 另一个列表项
[链接文本](https://example.com)
```

## 关键优势

- **易学** — 5 分钟即可上手
- **纯文本** — 可使用任何编辑器打开
- **可移植** — 可转换为 HTML、PDF、DOCX 等多种格式
- **AI 友好** — 所有主流 AI 聊天工具都使用 Markdown 输出
