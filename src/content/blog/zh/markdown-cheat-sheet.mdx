# Markdown 速查表 — 快速参考指南

收藏本页。你需要的所有 Markdown 语法，附带示例。

## 基础语法

```markdown
# 一级标题
## 二级标题
### 三级标题

**加粗文本**
*斜体文本*
~~删除线~~

[链接文本](https://example.com)
![图片替代文本](image-url.jpg)

> 引用
```

## 列表

```markdown
- 无序列表项 1
- 无序列表项 2
  - 嵌套列表项

1. 有序列表项 1
2. 有序列表项 2

- [ ] 待办事项
- [x] 已完成事项
```

## 代码

````markdown
`行内代码`

```javascript
// 带语法高亮的代码块
const greeting = "Hello, world!";
console.log(greeting);
```
````

## 表格

```markdown
| 表头 1   | 表头 2    | 表头 3   |
|----------|:--------:|---------:|
| 左对齐   | 居中对齐  | 右对齐   |
| 数据     | 数据     | 数据     |
```

效果：

| 表头 1   | 表头 2    | 表头 3   |
|----------|:--------:|---------:|
| 左对齐   | 居中对齐  | 右对齐   |
| 数据     | 数据     | 数据     |

## 链接与引用

```markdown
[基础链接](https://example.com)
[带标题的链接](https://example.com "标题")

[引用式链接][ref]

[ref]: https://example.com
```

## 图片

```markdown
![替代文本](image.jpg)
![替代文本](image.jpg "标题")
```

## 水平分隔线

```markdown
---
```

## Markdown 中使用 HTML

Markdown 支持原生 HTML：

```html
<details>
<summary>点击展开</summary>
隐藏内容在此。
</details>
```
