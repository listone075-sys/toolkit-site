# 如何在 Markdown 中创建表格

手写 Markdown 表格是一件繁琐的事情。一个管道符写错，整个表格就坏了。下面介绍正确的语法——以及更简单的方法。

## 基础表格语法

```markdown
| 姓名   | 年龄 | 城市         |
|--------|------|--------------|
| 张三   | 30   | 北京         |
| 李四   | 25   | 上海         |
| 王五   | 28   | 深圳         |
```

渲染效果：

| 姓名   | 年龄 | 城市         |
|--------|------|--------------|
| 张三   | 30   | 北京         |
| 李四   | 25   | 上海         |
| 王五   | 28   | 深圳         |

## 列对齐

```markdown
| 左对齐    | 居中对齐  |     右对齐 |
|:----------|:---------:|----------:|
| `:---`    | `:---:`   | `---:`    |
| 左对齐 L  | 居中 C    | 右对齐 R  |
```

## 表格技巧

- **表头行是必须的**——每个 Markdown 表格都需要列标题
- **对齐行是必须的**——第二行（`|---|---|`）用于定义对齐方式
- **转义管道符**——使用 `\|` 在单元格中包含字面管道符
- **列数一致**——每行必须有相同数量的列

## 常见错误

```markdown
❌ | A | B | C |         ← 表头
   |---|---|             ← 少了一列！
   | 1 | 2 | 3 |

✅ | A | B | C |
   |---|---|---|
   | 1 | 2 | 3 |
```
