# 如何将 PNG 转换为 JPG（以及何时应该转换）

PNG 在截图和 Logo 方面表现完美。但对于照片来说，PNG 文件可能是 JPEG 的 5–10 倍大，却没有任何肉眼可见的画质提升。以下是何时转换以及如何转换。

## 何时将 PNG 转换为 JPG

| 场景 | 转换？ | 原因 |
|----------|----------|-----|
| 以 PNG 格式保存的照片 | ✅ 是 | PNG 存储照片体积大 5–10 倍 |
| 包含文字的截图 | ❌ 否 | JPEG 会让文字变模糊 |
| 带透明背景的 Logo | ❌ 否 | JPEG 不支持透明 |
| 用于邮件的照片 | ✅ 是 | 文件更小 = 发送更快 |
| 用于网站的照片 | ✅ 是 | 页面加载更快 |
| 电商产品照片 | ✅ 是 | 大多数平台偏好 JPEG |

## 如何转换

1. 打开 ToolCraft 上的任意图片工具 —— [WebP 转 JPG 转换器](https://toolcraftbox.com/tools/webp-to-jpg)同样支持 PNG
2. 上传你的 PNG 文件
3. 转换在你浏览器中完成
4. 以 JPEG 格式下载

## 画质对比

原始文件：
`photo.png` — 4.2 MB

转换后：
`photo.jpg` — 820 KB（缩小 80%）

在 90% 质量下，人眼几乎无法察觉差异。你的网站加载更快，邮件发送更迅速，还能节省存储空间。

## 专业技巧：批量转换

如果你有多张 PNG 照片，可以逐张转换。由于所有处理都在本地完成，没有排队等待 —— 每次转换只需几秒钟。
