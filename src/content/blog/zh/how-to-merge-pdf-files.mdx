# 如何在线合并 PDF 文件（免费、无需注册）

将多个 PDF 文件合并为一个文档，不应该需要昂贵的软件，也不应该需要把敏感文件上传到未知服务器。

## 快速步骤

1. 前往 [合并 PDF 工具](/tools/merge-pdf)
2. 拖放所有 PDF 文件（或点击浏览选择）
3. 按正确顺序排列它们
4. 点击"合并 PDF"
5. 下载合并后的文档

## 为什么选择我们的工具？

- **隐私保护**：所有处理都在你的浏览器中完成。你的 PDF 绝不会上传到任何服务器。
- **极速体验**：无需等待上传 — 合并在你的设备本地进行。
- **无限制**：想合并多少 PDF 就合并多少，想用多少次就用多少次。
- **完全免费**：无需注册、无需试用、无需信用卡。

## 常见使用场景

- 将扫描的页面合并为一个文档
- 合并报告的各个章节
- 整理发票或收据
- 组装作品集文档
