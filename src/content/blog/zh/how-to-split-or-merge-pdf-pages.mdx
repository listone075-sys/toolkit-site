# 如何拆分和合并 PDF 页面 — 完整指南

你有一份 50 页的 PDF，但你只需要第 3 到第 7 页。

或者你有五份独立的 PDF — 发票、报告和附录 — 需要把它们合成一份干净的文档。

这些不是极端场景，而是数百万人的日常工作。而且你不需要 Adobe Acrobat 就能搞定。

以下是如何免费、直接在浏览器中拆分和合并 PDF — 无需上传、无需注册、无需安装软件。

## 拆分 PDF：提取你需要的部分

### 什么时候需要拆分 PDF？

- **发送单个章节** — 从 200 页的报告中提取第 12-24 页
- **分离发票** — 30 张发票的合并 PDF → 30 个独立文件
- **删除空白页** — 拆分，删除空白页，合并剩余的
- **仅分享相关页面** — 避免不小心发送保密页面
- **缩小文件体积** — 将大 PDF 拆分成小份以便邮件发送

### 方法一：拆分所有页面

将每一页变成独立的 PDF 文件：

1. 打开 **[PDF 拆分工具](/tools/split-pdf)**
2. 上传你的 PDF（拖拽或点击选择文件）
3. 点击**拆分所有页面**
4. 下载单独页面 — 每个文件以原始页码命名

当你需要重新组织文档或提取特定页面用于其他地方时，这种方法最合适。

### 方法二：提取页码范围

只需要大文档中的第 5-10 页？

1. 打开 **[PDF 拆分工具](/tools/split-pdf)**
2. 上传你的 PDF
3. 在范围输入框中输入页码范围（如 `5-10`）
4. 点击提取 — 得到一个只包含这些页面的 PDF

**页码范围示例：**

| 你想要 | 你输入 |
|--------|--------|
| 第 3 到 7 页 | `3-7` |
| 只要第 1 页 | `1-1` |
| 第 10 页到末尾 | `10-` |
| 前 5 页 | `-5` |

### 方法三：按大小拆分

需要把大 PDF 拆成小份以便邮件发送？先拆分所有页面，再用 **[PDF 合并工具](/tools/merge-pdf)** 将页面按 5 页或 10 页分组合并。详见下文。

## 合并 PDF：把所有内容合成一个文件

### 什么时候需要合并 PDF？

- **月度报告** — 将周报合并为月度文档
- **求职申请** — 把简历 + 求职信 + 作品集合并成一份 PDF
- **发票汇总** — 将所有发票作为一个文件发给财务
- **会议准备** — 把议程 + 报告 + 幻灯片合并成一份简报包
- **学习资料** — 按主题合并讲义、幻灯片和阅读材料

### 如何合并 PDF（30 秒搞定）

1. 打开 **[PDF 合并工具](/tools/merge-pdf)**
2. 上传所有要合并的 PDF 文件
3. **拖拽排列顺序** — 第一个文件变成第 1 页
4. 点击**合并 PDF**
5. 下载合并后的 PDF

工具会保留原文件的一切 — 字体、图片、格式、链接和页面尺寸。即使你的 PDF 页面尺寸不同（A4 和 Letter 混合），每页也保持原始尺寸。

### 合并技巧

- **命名很重要** — 上传前重命名文件（如 `01-封面.pdf`、`02-报告.pdf`、`03-附录.pdf`），这样上传时顺序就是对的
- **合并前预览** — 确认所有文件都在且顺序正确
- **分批合并** — 合并 50+ 个 PDF？先 10-15 个一组合并，再合并各组。更容易发现排序错误。

## 常见问题

**Q: 有文件大小限制吗？**
A: 没有硬性限制 — 处理都在浏览器中进行。非常大的 PDF（200MB+）可能因设备性能而变慢。为获得最佳体验，文件尽量控制在 100MB 以内。

**Q: 能拆分有密码保护的 PDF 吗？**
A: 拆分工具适用于未加密的 PDF。请先移除密码保护，然后再拆分。

**Q: 合并会改变格式吗？**
A: 不会。合并工具使用 PDF-lib，精确保留原 PDF 的字体、图片、布局和链接。

**Q: 能合并不同页面尺寸（A4 + Letter）的 PDF 吗？**
A: 可以。每页在合并输出中保持其原始尺寸，不会强制统一大小。

**Q: 如何删除 PDF 中的特定页面？**
A: 拆分所有页面 → 只下载你想保留的页面 → 可选：将它们合并回一份 PDF。

**Q: 能一次性合并 100 个 PDF 吗？**
A: 技术上可以，但对于大批量合并，建议分 10-15 个一组进行。这样更容易在最终合并前验证顺序是否正确。

**Q: 还有其他 PDF 操作吗？**
