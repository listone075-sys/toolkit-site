# How to Create Tables in Markdown

Writing tables in Markdown by hand is tedious. One wrong pipe character and it breaks. Here's the syntax — and a better way.

## Basic Table Syntax

```markdown
| Name  | Age | City          |
|-------|-----|---------------|
| Alice | 30  | New York      |
| Bob   | 25  | San Francisco |
| Carol | 28  | Chicago       |
```

Renders as:

| Name  | Age | City          |
|-------|-----|---------------|
| Alice | 30  | New York      |
| Bob   | 25  | San Francisco |
| Carol | 28  | Chicago       |

## Column Alignment

```markdown
| Left      | Center    |     Right |
|:----------|:---------:|----------:|
| `:---`    | `:---:`   | `---:`    |
| Aligns L  | Aligns C  | Aligns R  |
```

## Table Tips

- **Header row is required** — every Markdown table needs column headers
- **Alignment row is required** — the second row (`|---|---|`) defines alignment
- **Escaping pipes** — use `\|` to include a literal pipe character in a cell
- **Consistent columns** — every row must have the same number of columns

## Common Mistakes

```markdown
❌ | A | B | C |         ← Header
   |---|---|             ← Missing column!
   | 1 | 2 | 3 |

✅ | A | B | C |
   |---|---|---|
   | 1 | 2 | 3 |
```
