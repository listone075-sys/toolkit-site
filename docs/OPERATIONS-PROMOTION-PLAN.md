#### SEO 清单（每篇必查）

- [ ] 标题包含主关键词（前60字符）
- [ ] Meta description 包含关键词+CTA（140-160字符）
- [ ] URL slug 简洁且包含关键词
- [ ] H1 唯一且包含关键词
- [ ] H2/H3 结构分层清晰
- [ ] 首段在100字内点明主题+关键词
- [ ] 图片有描述性 alt 文本
- [ ] 至少2个内部链接到相关工具页面
- [ ] 至少1个内部链接到相关博客文章
- [ ] 至少1个外部链接到权威来源
- [ ] 包含 FAQ 区块（可获 FAQ Schema）
- [ ] 结尾有明确的 CTA（试用工具 / 订阅 / 分享）

### 3.2 关键词研究流程

**工具**: Google Keyword Planner（免费）, Ahrefs Webmaster Tools（免费）, Google Trends

**步骤**:
1. 从工具名称出发，用 Google Suggest 扩展 → 收集 "people also ask" 问题
2. 在 Ahrefs KW Explorer 验证搜索量和关键词难度（优先 KD < 20 的词）
3. 在 Google Trends 检查趋势走向
4. 分析搜索结果第一页内容类型和质量（是否有内容缺口）
5. 确定目标关键词和内容策略

**筛选标准**:
- 搜索量: 50-5000（太低没流量，太高竞争不过）
- KD (Keyword Difficulty): < 30（6个月内有望排到第一页）
- 商业/交易意图优先于纯信息意图
- 与本站工具直接相关的优先
